## 3. ATR动态止盈止损算法

### 计算公式
```
TR = max(当前高-低, |高-前收|, |低-前收|)  
ATR(14) = (ATR前×13 + TR当前)/14  
ATR基准 = 30周期ATR均值
```

### 波动系数
- 标准(0.8-1.2倍基准)：1.0  
- 高波动(>1.2)：1.2（放宽止盈止损空间）  
- 低波动(<0.8)：0.8（收紧阈值）  
- 极端(>2.0)：止损系数0.6，禁止开新仓，严密监控  

### 动态止盈止损阈值
```
止盈1 = max($0.10, 0.5×ATR×波动系数)  
止盈2 = max($0.15, 0.75×ATR×波动系数)  
止盈3 = max($0.20, 1.0×ATR×波动系数)  

止损1单持 = -max($0.05, 0.3×ATR×波动系数)  
止损1多持 = -max($0.07, 0.25×ATR×波动系数)  
止损2 = -max($0.10, 0.6×ATR×波动系数)  
止损3 = -max($0.15, 1.0×ATR×波动系数)
```

## 4. 持仓管理核心规则（方向单一·3仓位版）

### 持仓数量管理
```
- 单持仓默认最多3个  
- 禁止超过3个持仓  
- 所有持仓方向必须一致（全多或全空）
```

### 方向单一铁律
- 所有持仓必须同向  
- 禁止多空对冲同时存在  
- 单持仓方向自由  
- 多持仓新仓必须与已有持仓同向  

### 多持仓入场条件（均需全部满足）

**第2个持仓入场条件**  
1. 已有1持仓且浮盈>0  
2. 保证金率<12%  
3. 新仓方向与已有持仓同向  
4. 信号极强（4/4指标确认）  
5. 两币相关性<0.6  
6. 不同板块  
7. 账户无连续亏损  
8. 非冷静期或紧急状态  
9. 风险回报≥1:3  
10. 已有持仓周期<12  

**第3个持仓入场条件**  
1. 已有2持仓且均浮盈>0  
2. 保证金率<8%  
3. 新仓方向与已有持仓同向  
4. 信号极强（4/4指标确认）  
5. 三币相关性均<0.5  
6. 三币不同板块  
7. 账户无连续亏损  
8. 账户历史最大回撤>-0.25  
9. 市场波动率<历史均值×1.3  
10. 已有持仓周期均<10  
11. 风险回报≥1:3.5  
12. 非冷静期或紧急状态  

### 持仓周期管理
- 单持仓强制平仓周期：15  
- 延期：16-17（满足浮盈、技术强势、市场环境、风险管理4条件）  
- 绝对上限：18周期必须平仓  
- 双持仓周期和≤20，三持仓周期和≤25  
- 禁止所有持仓同时接近15周期  

### 风险叠加系数  
- 单持仓风险系数1.0  
- 双持仓风险系数2.2  
- 三持仓风险系数3.5

## 5. 方向切换管理机制

### 切换触发条件（满足任一）
- 技术指标全面反转（RSI、MACD、EMA交叉）  
- 市场结构性转变（≥3币种反向信号、极端情绪）  
- 持仓连续亏损（连续3个持仓亏损、累计亏损>-0.30且技术转向）  

### 切换流程
1. 全平当前方向持仓  
2. 强制冷静期3-5周期，不允许反向入场  
3. 技术指标4/4确认后方允许反向开仓  
4. 记录切换决策与反思  

### 风险控制
- 同币切换间隔≥10周期  
- 整体方向切换间隔≥5周期  
- 切换后首仓减半，止损更严格(-0.03)  
- 2周期内回转触发紧急撤回，恢复冷静期

### 多持仓风险监控（每周期）
- 方向全一致  
- 超限立即减仓  
- 至少50%持仓浮盈  
- 浮亏累计>0.10需评估减仓  
- 技术衰退及时减仓  
- 持仓周期超限优先平最长周期仓位  
- 市场异常立即减少为单持仓  

### 紧急规则（立即执行）
- 日跌幅>-0.40，全平  
- 所有币RSI<40且MACD全负，全平  
- 单仓持仓周期≥18，立即平仓  
- 连续亏损≥3周期，全平  
- 多持仓保证金超限，立即减仓  
- 连续5周期无操作，系统暂停

- 止损后：3周期  
- 紧急风险：3-5周期

### 轮换规则

- 同币轮换间隔≥2周期  
- 禁止频繁切换
