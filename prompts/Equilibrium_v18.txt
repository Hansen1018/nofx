## 9. 止损止盈计算（完整版）

### 9.1 开多（LONG）止损止盈计算

```python

Step 3: 确定情绪系数

    emotion_coefficient = 0.8
else:
    emotion_coefficient = 1.0

Step 4: 计算止损距离

Step 5: 计算止损价格
stop_loss_price = current_price - stop_loss_distance
stop_loss = round(stop_loss_price, decimal_places)

# ===== 第二部分：止盈计算（3倍风险回报比）=====

Step 6: 计算风险金额
risk_amount = abs(current_price - stop_loss)

Step 7: 计算止盈距离（3倍风险）
take_profit_distance = risk_amount × 3.0

Step 8: 计算止盈价格
take_profit_price = current_price + take_profit_distance
take_profit = round(take_profit_price, decimal_places)

# ===== 第三部分：边界验证 =====

Step 9: 最小止损检查（≥0.3%）
min_stop_loss_percentage = 0.003
actual_stop_loss_percentage = abs(stop_loss - current_price) / current_price

if actual_stop_loss_percentage < min_stop_loss_percentage:
    # 调整至最小0.3%
    stop_loss_distance = current_price × min_stop_loss_percentage
    stop_loss_price = current_price - stop_loss_distance
    stop_loss = round(stop_loss_price, decimal_places)
    
    # 重新计算止盈
    risk_amount = abs(current_price - stop_loss)
    take_profit_distance = risk_amount × 3.0
    take_profit_price = current_price + take_profit_distance
    take_profit = round(take_profit_price, decimal_places)

# ===== 第四部分：强制验证（必须通过）=====

Step 10: 多重验证确保数据有效性

# 验证1：止损止盈必须>0
if stop_loss <= 0 or take_profit <= 0:
    ❌ 拒绝开仓

# 验证2：开多时止损必须<当前价
if action == "open_long" and stop_loss >= current_price:
    ❌ 拒绝开仓

# 验证3：开多时止盈必须>当前价
if action == "open_long" and take_profit <= current_price:
    ❌ 拒绝开仓

# 验证4：止损距离不能过小（最小0.3%）
# 验证5：止盈距离不能过小（最小0.9%）
```
