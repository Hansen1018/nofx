## 4. 六层止损系统

### Layer 1：快速止损

- 浮亏**-0.6%**且5分钟RSI连降5点+MACD死叉 → 立即平仓
- 持仓4周期内浮亏扩大>**0.35%** → 立即平仓
- 15分钟与1小时背离+浮亏**-1.0%** → 立即平仓


### Layer 2：ATR动态止损

```
止损阈值 = -max(0.05U, 0.3×ATR×波动系数)
波动系数：评分>70用1.2 | 50-70用1.0 | <50用0.8
```


### Layer 3：时间止损

**第6周期**：浮亏任意值 → 橙色预警

**第10周期强制检查**：

1. 若浮盈曾触发Layer 4 Level 1（+0.3%）→ 按Layer 4规则执行
2. 若浮盈从未触发Level 1 → 浮亏任意值立即平仓，浮盈0-0.3%立即平仓
3. 浮盈0.3-1.0% → 移动止损至保本+0.15%，延长至14周期
4. 浮盈1.0-2.0% → 平仓50%，剩余延长至18周期
5. 浮盈>2.0% → 执行Level 3分批止盈，延长至22周期

### Layer 4：浮盈保护系统

**Level 1 - 保本锁定（+0.3%触发）**：

- 硬止损：开仓价+0.1%
- 软预警：+0.18%（回落至此立即平仓50%）

**Level 2 - 动态追踪（+0.8%触发）**：

- 追踪距离：0.45%（为40秒延迟预留0.15%）
- 高波动：放宽至0.55%
- RSI转向：收紧至0.3%

**Level 3 - 分批锁利（+1.8%触发）**：

- 立即平仓50%
- 剩余50%止损移至+0.8%
- 从+1.8%回撤至+1.0% → 平仓剩余全部


### Layer 5：市场环境止损

评分<30时：止损收紧至-0.5%

### Layer 6：极端波动熔断

5分钟内价格波动>2% → 浮亏任意值立即平仓
