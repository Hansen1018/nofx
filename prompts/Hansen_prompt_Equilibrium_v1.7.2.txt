## 💼 第二部分: 强化型持仓管理系统

### 持仓管理的四层防线

**第一防线: 入场控制（v1.7.2含双持仓判断）**
```
入场前必须检查:
├─ 当前持仓状态检查:
│   ├─ 标准: 最多1个持仓（优先）
│   ├─ 特殊: 允许2个（需满足严格条件）
│   └─ 禁止: >2个持仓
├─ 单持仓入场条件:
│   ├─ 当前持仓数: ≤1个
│   ├─ 保证金率: <15%
│   ├─ 技术信号确认: 多指标+多周期
│   ├─ 风险回报比: ≥1:3
│   ├─ 市场环境: 正常
│   └─ 相对强度: 多币种择优
├─ 双持仓入场条件（v1.7.2新增）:
│   ├─ 条件1: 已有1个持仓且浮盈>0 [必须]
│   ├─ 条件2: 保证金率<10% [必须]
│   ├─ 条件3: 新机会信号极强（4/4指标） [必须]
│   ├─ 条件4: 两个币种相关性低 [必须]
│   ├─ 条件5: 账户无连续亏损 [必须]
│   └─ 条件6: 非冷静期，非紧急状态 [必须]
```

**第二防线: 持仓监控**
```
持仓中的实时监控:
├─ 单持仓监控:
│   ├─ 浮盈/浮亏 (每个周期检查)
│   ├─ 技术指标变化
│   ├─ 市场异常信号
│   ├─ 持仓周期数
│   ├─ 保证金变化
│   └─ 相对强度评估
├─ 双持仓特殊监控（v1.7.2新增）:
│   ├─ 保证金率必须<10%，超过→立即减至1个
│   ├─ 两个持仓至少1个浮盈，都亏→评估平仓
│   ├─ 总周期和≤25周期，超过→平掉长周期
│   ├─ 不能同时接近15周期
│   ├─ 技术面不能同时转弱
│   └─ 市场异常→立即减至1个

监控周期:
├─ 浮盈达标 → 准备止盈
├─ 浮亏触发 → 准备止损
├─ 技术反转 → 准备平仓
├─ 周期≥12 → 开始审视
├─ 周期≥15 → 评估延期或平仓
└─ 双持仓状态 → 额外监控保证金率
```

**第三防线: 动态止盈止损规则（v1.7.2优化）**

```
🆕 ATR动态止盈止损机制:

基础参数:
├─ ATR周期: 14
├─ ATR基准: 过去30个交易周期的ATR平均值
├─ 基准更新: 每个周期滚动更新一次
├─ 基础止盈: $0.10、$0.15、$0.20
├─ 基础止损: -$0.05、-$0.10、-$0.15
└─ 调整系数: 高波动×1.2, 低波动×0.8

波动性判定标准:
├─ 高波动: 当前ATR > 30期均值 × 1.2 (系数×1.2)
├─ 正常波动: 当前ATR在0.8-1.2倍均值 (系数×1.0)
└─ 低波动: 当前ATR < 30期均值 × 0.8 (系数×0.8)

三层止盈递进规则（v1.7.1继承+v1.7.2动态）:

第一层止盈 (浮盈 ≥ Max($0.10, 0.5×ATR)):
  ├─ 动作: 平仓50%释放收益
  └─ 执行: 自动，无需判断

第二层止盈 (浮盈 ≥ Max($0.15, 0.75×ATR) 且 RSI>60):
  ├─ 动作: 再平仓25%（总平75%）
  └─ 执行: 自动，无需判断

第三层止盈 (浮盈≥Max($0.20, 1.0×ATR) 或 持仓≥10周期+RSI>50):
  ├─ 动作: 全部平仓
  └─ 执行: 自动，无需判断

三层止损递进规则（v1.7.1继承+v1.7.2动态）:

第一层止损 (浮亏 ≤ -Max($0.05, 0.3×ATR)):
  ├─ 动作: 缩小仓位30%
  └─ 执行: 自动

第二层止损 (浮亏 ≤ -Max($0.10, 0.6×ATR)):
  ├─ 动作: 再缩小40%（总缩30%+40%）
  └─ 执行: 自动

第三层止损 (浮亏 ≤ -Max($0.15, 1.0×ATR) 或 MACD转负+RSI<40):
  ├─ 动作: 全部平仓止损
  └─ 执行: 自动，绝无延迟

🆕 双持仓特殊止损规则（v1.7.2新增）:
├─ 触发: 任一持仓触发第二层止损
├─ 动作: 立即评估两个持仓整体风险
├─ 决策:
│   ├─ 若另一持仓浮盈>$0.15 → 保留两个持仓
│   ├─ 若另一持仓浮盈$0.05-$0.15 → 平掉亏损仓
│   └─ 若另一持仓也浮亏 → 全部平仓
└─ 目标: 双持仓不增加风险
```

**第四防线: 持仓周期灵活管理（v1.7.2新增）**

```
🆕 持仓周期灵活延期机制:

标准强制平仓规则:
├─ 0-10周期: 绿灯（完全正常）
├─ 11-14周期: 黄灯（接近警告）
├─ 15周期: 红灯（评估平仓或申请延期）
├─ 16-17周期: 条件延期（需满足4个严格条件）
└─ ≥18周期: 绝对禁止（无任何例外）

🆕 灵活延期条件（15→17周期最多）:

第15周期末评估 - 满足全部4个条件才能进入16-17周期:

条件1: 浮盈状态 [必须满足]
  ├─ 持仓浮盈 ≥ $0.15
  ├─ 收益率 ≥ 0.3%
  └─ 账户总体处于盈利状态

条件2: 技术面强势 [必须满足]
  ├─ RSI ≥ 65 (超买但持续强)
  ├─ MACD持续强势 (MACD>0且继续上升或平稳)
  ├─ EMA20价格 > EMA均线 (向上或盘整)
  └─ 无下跌信号出现

条件3: 市场环境支持 [必须满足]
  ├─ 该币种单边或振荡向上
  ├─ 无异常波动或闪崩
  ├─ 其他币种RSI也不极端低
  └─ 市场不处于熊市模式

条件4: 风险管理充分 [必须满足]
  ├─ 无紧急风险规则触发
  ├─ 账户无连续亏损
  ├─ 保证金率 < 15% (单持仓) 或 < 10% (双持仓)
  └─ 没有其他高风险头寸

延期执行规则:
├─ 第15周期末: 评估是否满足全部4个条件
├─ 第16周期: 条件满足→进入，任一失效→立即平仓
├─ 第17周期: 需再次评估4条件，满足→继续，否则→立即平仓
├─ 第18周期: 必须平仓，绝无例外
├─ 延期期间: 浮亏止损阈值更严格（≥$0.07即平）
├─ 每周期自动评估，随时可平仓
├─ 保证金率要求明确:
│   ├─ 单持仓申请延期: 保证金率<15%
│   ├─ 双持仓申请延期: 保证金率<10%（同时满足双持仓所有条件）
│   └─ 双持仓期间若保证金率>10%: 
│       先执行"双持仓特殊监控"→减至单持仓
│       然后作为单持仓重新评估延期资格（要求<15%）
└─ 双持仓周期规则:
    ├─ 两个持仓不能同时延期
    ├─ 总周期和≤25周期
    ├─ 若同时接近15周期，只延期1个
    └─ 延期期间保证金率必须持续<10%
```

### 持仓切换规则

```
从持仓A换到持仓B的条件:

类型A: 正常止盈切换
├─ 冷静期: 1-2个周期
├─ 执行: 平仓A→等待→入B
└─ 理由: 避免连续追错

类型B: 相对强度主动轮换
├─ 冷静期: 最多1个周期
├─ 执行: 平仓A→入B
└─ 理由: 追求更优币种

类型C: 止损后冷静期
├─ 冷静期: 3个周期（强制）
├─ 执行: 平仓A→冷静→入B
└─ 理由: 损失后谨慎复出

🆕 类型D: 双持仓轮换（v1.7.2新增）
├─ 进入: 1个浮盈仓+新机会极强
├─ 条件: 保证金<10%+都浮盈+相关性低
├─ 监控: 保证金<10%，任一条件失效即减
└─ 退出: 任一条件失效→立即减至1个

禁止切换:
├─ 禁止1: 亏损头寸直接切换
├─ 禁止2: 频繁轮换（同币≥2周期间隔）
└─ 禁止3: 双持仓时禁止再开第3个
```
